# PART 7: TROUBLESHOOTING

### Low Engagement?
- Test different images
- Adjust targeting (broader or narrower)
- Try different ad copy
- Test different CTAs

### High CPM?
- Audience too narrow - expand interests
- Creative fatigue - rotate images
- Increase budget for more delivery

### Few Messages?
- CTA not clear enough - simplify keyword
- Add more urgency
- Test different keyword triggers
- Make the value proposition clearer

### Low Response Rate?
- Automated messages too long
- Not enough personalization
- Missing clear next step

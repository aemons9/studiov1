## How to Upload to Behance

### For Each Project:

1. Go to behance.net and click "Create a Project"
2. **Title**: Copy from PROJECT.md `## Project Title` section
3. **Cover Image**: Upload from `assets/covers/` folder
4. **Project Content**:
   - Copy description text from PROJECT.md
   - Upload images from project's `images/` folder
   - Add text blocks between images as specified
5. **Creative Fields**: Select tags from PROJECT.md
6. **Tags**: Copy comma-separated tags from PROJECT.md
7. **Settings**:
   - Enable "Allow people to appreciate this project"
   - Enable "Show in galleries"
   - For Project 06: Enable "Mature Content"

### Profile Setup:

1. Navigate to profile settings
2. Follow all instructions in `profile/PROFILE_SETUP.md`
3. Upload profile photo and banner
4. Complete all bio sections
